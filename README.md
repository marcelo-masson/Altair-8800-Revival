# Altair 8800 Revival
A personal learning project focused on the Altair 8800.

This repository documents my journey exploring early microcomputers, low-level programing, and computer architecture.

## 🎯 Goals
- Understand how the Altair 8800 works

- Learn Intel 8080 assembly language

- Study memory organization and addressing

- Explore how early computers executed programs

## 📚 What you'll find here
- Notes and study materials

- Assembly code examples (Intel 8080)

- Memory analysis and experiments

- Simulations and tests

## 💡 Why this project?
The Altair 8800 is one of the first personal computers.

Studying it helps build a deep understanding of how computers actually work at a low level.

Work in progress --- This repository will grow as I continue learning.
