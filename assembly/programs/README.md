# Altair 8800 Programs
This folder contains functional programs for the Altair 8800 (Intel 8080 CPU).

Since original assembly files are rare and often entered manually via front panel switches, I have provided a detailed breakdown of each program.

## How to Read These Programs
To bridge the gap between vintage hardware and modern programming, each program follows this conversion table:
* **Steps:** This is self-explanatory. It's simply the sequence of steps in the program.
* **Mnemonic:** The instruction that represents the machine language code.
* **Binary:** For physical switch toggling (grouped in 3s for the Altair panel).
* **Octal:** The standard numbering system used in the 1975 manuals.
* **Hexadecimal:** For the modern emulators and 8080 assembly debuggers.
* **Decimal:** I added this for personal preference, just to visualize the decimal equivalent.

This implementation is based on the **original Altair 8800 Operator's Manual**.
